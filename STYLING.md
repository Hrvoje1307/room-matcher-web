# Styling System Reference

This project uses **Panda CSS** — a build-time, type-safe CSS-in-JS library.  
All generated code lives in `styled-system/` (auto-generated, never edit directly).  
Run `npm run dev` or `npm run prepare` to regenerate after config changes.

---

## Import paths

```ts
// JSX primitives — Box, styled, and all HTML element factories
import { Box, styled } from "../../styled-system/jsx";

// css() utility — produces a className string from a style object
import { css } from "../../styled-system/css";

// cva() — compound variant helper (for building recipe-like components inline)
import { cva } from "../../styled-system/css";

// cx() — merges classNames (like clsx)
import { cx } from "../../styled-system/css";

// Layout patterns — ready-made flex/grid shorthand components
import { Flex, Grid, Stack, HStack, VStack, Center, Wrap } from "../../styled-system/jsx";

// Recipes — generated from theme/recipes/
import { button } from "../../styled-system/recipes";
```

**Path alias:** `@/*` → `src/*`. The styled-system folder is at the repo root, so imports from `src/` always go `../../styled-system/...` (two levels up).

## Core usage

### Box — the base building block

`Box` is a `div` that accepts a `css` prop with type-safe style tokens.

```tsx
<Box css={{ display: "flex", gap: "16px", backgroundColor: "cream.500" }}>
  content
</Box>
```

### styled — factory for any HTML element

```tsx
import { styled } from "../../styled-system/jsx";

// Inline style via css prop
<styled.p css={{ fontSize: "14px", color: "gray.600" }}>text</styled.p>
<styled.h1 css={{ fontWeight: "700", color: "navy.500" }}>heading</styled.h1>
<styled.input css={{ borderRadius: "12px", padding: "12px 16px" }} />
<styled.button css={{ cursor: "pointer" }} />
<styled.select css={{ appearance: "auto" }} />
<styled.label css={{ textTransform: "uppercase" }} />
```

### css() — generates a className string

Use this when you need a `className` prop (e.g. Next.js `<Link>`, native `<input>`).

```tsx
import { css } from "../../styled-system/css";

<Link href="/" className={css({ color: "coral.500", _hover: { color: "coral.600" } })}>
  link
</Link>

<input className={css({ width: "100%", borderRadius: "12px" })} />
```

### Built-in colors

Panda CSS also includes the full Tailwind color palette (`rose`, `pink`, `sky`, `blue`, `red`, etc.) plus `white`, `black`, `transparent`, `current`.

```ts
backgroundColor: "white"
color: "red.500"    // Tailwind red (for errors)
```

### token() helper — use inside string values

```ts
boxShadow: "0 0 0 3px token(colors.coral.100)"
```

## Responsive values

Mobile-first. Use an object with breakpoint keys:

```ts
// base = all screens, md = ≥768px, lg = ≥1024px
flexDirection: { base: "column", md: "row" }
display: { base: "none", md: "flex" }
fontSize: { base: "24px", md: "32px", lg: "40px" }
```

Available breakpoints: `sm`, `md`, `lg`, `xl`, `2xl`

## Layout patterns

These are shorthand components generated by Panda. They accept CSS props directly (no `css={}` wrapper needed).

```tsx
import { Flex, Grid, Stack, HStack, VStack, Center } from "../../styled-system/jsx";

<Flex align="center" justify="space-between" gap="16px">...</Flex>

<Grid columns={3} gap="24px">...</Grid>

<Stack gap="12px">...</Stack>       // vertical stack
<HStack gap="8px">...</HStack>      // horizontal stack
<VStack gap="8px">...</VStack>      // vertical stack alias

<Center h="100vh">...</Center>
```

## Button recipe

Defined in `src/theme/recipes/button.recipe.ts`, imported via:

```ts
import { button } from "../../styled-system/recipes";
// or use the component:
import { Button } from "@/shared/components/ui/button";
```

### Variants

| `variant` | Description |
|-----------|-------------|
| `primary` | coral bg, white text |
| `secondary` | navy bg, white text |
| `outline` | transparent bg, gray border |
| `ghost` | transparent bg, navy text |
| `white` | white bg, navy text |

| `size` | padding | font |
|--------|---------|------|
| `sm` | px18 / py9 | 14px |
| `md` | px24 / py12 | 14px |
| `lg` | px28 / py14 | 18px |

Default: `variant="primary" size="md"`

```tsx
<Button variant="primary" size="md">Prijavi se</Button>
<Button variant="outline" size="sm">Odustani</Button>

// One-off style overrides — use style prop, not new recipe variants
<Button style={{ width: "100%", borderRadius: "12px" }}>Submit</Button>
```

## Common patterns in this codebase

### Full-width input with error state

```tsx
<styled.input
  css={{
    width: "100%",
    backgroundColor: "white",
    border: "1px solid",
    borderColor: hasError ? "red.400" : "gray.400",
    borderRadius: "12px",
    padding: "12px 16px",
    fontSize: "14px",
    color: "navy.500",
    outline: "none",
    _placeholder: { color: "gray.500" },
    _focus: { borderColor: "coral.400", boxShadow: "0 0 0 3px token(colors.coral.100)" },
  }}
/>
```

### Uppercase label

```tsx
<styled.label css={{
  display: "block",
  fontSize: "11px",
  fontWeight: "600",
  letterSpacing: "0.08em",
  textTransform: "uppercase",
  color: "gray.600",
  marginBottom: "8px",
}} />
```

### Page background wrapper

```tsx
<Box css={{ backgroundColor: "cream.500", minHeight: "100dvh" }}>
```

### Fixed header

```tsx
<Box css={{ position: "fixed", top: "0", left: "0", right: "0", zIndex: "50", backgroundColor: "cream.500", borderBottom: "1px solid", borderColor: "gray.400" }}>
```

### Dynamic backgroundImage — MUST use style prop, not css prop

```tsx
// ✅ correct
<Box style={{ backgroundImage: `url(${imageUrl})` }} />

// ❌ wrong — Panda can't evaluate dynamic values at build time
<Box css={{ backgroundImage: `url(${imageUrl})` }} />
```

### Two CSS animations on same element — must be ONE declaration

```tsx
// ✅ correct — comma-separated in one animation property
css={{ animation: "fadeIn 0.3s ease, slideUp 0.4s ease" }}

// ❌ wrong — second overrides first, first animation never runs
css={{ animation: "fadeIn 0.3s ease" }}
css={{ animation: "slideUp 0.4s ease" }}
```

### position: sticky in flex container

```tsx
// sticky element MUST have alignSelf: flex-start or sticky won't work
<Box css={{ position: "sticky", top: "90px", alignSelf: "flex-start" }}>
```

## Spacing / sizing reference

Panda CSS uses raw values (no Tailwind spacing scale). Use pixel or rem strings directly:

```ts
padding: "16px"
gap: "24px"
marginBottom: "32px"
paddingX: "16px"    // shorthand for paddingLeft + paddingRight
paddingY: "20px"    // shorthand for paddingTop + paddingBottom
borderRadius: "12px"
fontSize: "14px"
```

## Keyframe animations

Define in component files using `@keyframes` inside `css` blocks, or via native `<style>` tags. Example from codebase:

```tsx
<style>{`
  @keyframes float {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-12px); }
  }
`}</style>
<Box css={{ animation: "float 3s ease-in-out infinite" }} />
```
